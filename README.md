# 河源职业技术学院校园网认证脚本 (HYPT Network Autologin Script)

## 简介

本项目是一个用于**河源职业技术学院**新版（2025年11月13日后启用）校园网认证的 Shell 脚本。

脚本设计运行在 [iStoreOS](https://site.istoreos.com/) / OpenWrt 或其他基于 Linux 的软路由系统中，通过 `curl` 命令模拟登录请求，实现网络认证的自动化，并能配合计划任务（cron）实现断线自动重连。

**本项目在 iStoreOS `24.10.4` 版本下测试通过。**

## 特性

- **自动认证**：无需手动打开登录页面，脚本自动完成认证流程。
- **状态监测**：基于 `mwan3` 状态进行精准判断，只在接口离线时才触发认证，减少不必要的请求。
- **多接口支持**：支持对多个 WAN 接口（如多拨场景）进行监控和认证。
- **日志记录**：输出清晰的执行日志，方便排查问题。
- **高兼容性**：使用 `sh` 编写，依赖 `curl`、`grep`、`sed` 等常用工具，在大多数 Linux 环境下都能运行。

## 使用方法

### 1. 准备工作

确保你的软路由系统已经安装了 `curl` 和 `mwan3` (如果需要基于 mwan3 判断)。iStoreOS 系统一般会自带。

### 2. 上传脚本

将 `login.sh` 脚本上传到你的软路由的任意目录下，例如 `/root/login.sh`。

### 3. 修改配置

打开 `login.sh` 脚本，找到顶部的 **配置区**，根据你的实际情况修改以下变量：

```sh
# ======================= 配置区 =======================
# 您的校园网账号和密码
USERNAME="你的学号"
PASSWORD="你的密码"

# 需要监控和认证的所有WAN接口，用空格隔开
# 注意：这里的名字必须和 mwan3 status 输出中的接口名完全一致
INTERFACES="wan" # 如果你有多拨，可以写成 "wan vwan1 vwan2"，用空格隔开每个逻辑端口
# ======================================================
```

**重要提示：** 请务必将 `USERNAME` 和 `PASSWORD` 修改为你自己的校园网账号和密码。

### 4. 授予执行权限

通过 SSH 连接到你的软路由，然后执行以下命令：

```sh
chmod +x /root/login.sh
```

### 5. 配置计划任务 (Crontab)

为了实现断线自动重连，需要让脚本每隔几分钟自动运行一次。

推荐通过 “系统” -> “计划任务” 的网页界面来添加，更加方便。在crontab配置中加入下面这行，代表每 2 分钟执行一次脚本：

```
*/2 * * * * /root/login.sh
```

保存并应用后，脚本就会按照你设定的频率自动检查网络状态并执行认证了。

## 调试

如果你发现脚本运行不正常，可以通过 SSH 手动执行以下命令来查看详细的调试输出：

```sh
sh /root/login.sh debug
```

## 技术实现简述

本脚本通过以下步骤模拟认证流程：
1.  首先访问学校指定的打开认证页面的 IP (`2.2.2.2`)，校园网认证系统会用返回的一个简单 html 页面，包含跳转到 Portal 登录页面 URL的 js 代码。
2.  脚本从 js 代码中获取重定向的 URL (`http://10.5.0.11/portal.do?...`) 并提取出 `wlanuserip`, `wlanacname`, `mac`, `vlan` 等关键参数。
3.  最后，脚本将提取出的参数与用户配置的账号密码组合，构造一个 GET 请求发送到认证接口 `http://10.5.0.11/quickauth.do`，从而完成认证。认证成功会返回包含 `"code":"0"` 和 `"message":"认证成功"` 等内容的 JSON 数据。
